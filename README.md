# Customer Segmentation Using K-Means Clustering

## 📌 Project Overview
This project demonstrates Customer Segmentation using the K-Means Clustering algorithm, an Unsupervised Machine Learning technique. The model groups customers based on their Annual Income and Spending Score to identify similar customer segments.

## 🎯 Objectives
- Understand the concept of Unsupervised Learning.
- Apply the K-Means Clustering algorithm.
- Segment customers based on their spending behavior.
- Visualize customer clusters using scatter plots.

## 📂 Dataset
**Dataset Name:** Mall Customers Dataset

### Features Used
- CustomerID
- Genre
- Age
- Annual Income (k$)
- Spending Score (1-100)

## 🛠️ Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🤖 Algorithm
- K-Means Clustering

## 📊 Workflow
1. Import the dataset.
2. Explore the dataset.
3. Select features.
4. Apply the Elbow Method.
5. Train the K-Means model.
6. Predict customer clusters.
7. Visualize the results.

## 📈 Output
The K-Means algorithm successfully grouped customers into five different clusters based on their Annual Income and Spending Score. These clusters help understand customer behavior and can support better marketing strategies.

## 🏆 Conclusion
This project demonstrates how Unsupervised Learning can discover hidden patterns in customer data. K-Means Clustering effectively segments customers into meaningful groups, which can be useful for business decision-making.
